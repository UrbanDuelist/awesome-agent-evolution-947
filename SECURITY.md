# Security Policy

Self Evolve is currently a static research and index site. Security issues are
most likely to involve unsafe links, exposed secrets, generated artifacts, or
accidentally published raw data.

## Report Scope

Please report:

- Exposed credentials or tokens.
- Malicious links or typosquatting in indexed resources.
- Raw data that should not be public.
- Build or deployment configuration that exposes private files.
- XSS or unsafe rendering in the website.

## Reporting

Open a private security advisory if the project is hosted on GitHub. If private
advisories are unavailable, open a minimal issue that says a security report is
needed, without including sensitive details.
